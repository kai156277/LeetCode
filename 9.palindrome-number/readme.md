# 整数反转法 (9.palindrome-number-1.cpp)
将数字本身反转，然后将反转后的数字与原始数字进行比较，如果它们是相同的，那么这个数字就是回文。

## 复杂度分析
* 时间复杂度：O(log 10 (n))， 对于每次迭代，我们会将输入除以10，因此时间复杂度为 O(log 10 (n))。
* 空间复杂度：O(1)。
